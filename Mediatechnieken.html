Beschrijving:

Voor deze opdracht moest ik een filmposter maken van een van mijn favoriete films, dit is uiteindelijk het resultaat.


Feedback:

Ontwerp 1

Bij het eerste ontwerp kreeg ik als feedback dat het font wat ik gebruikte te bold is, en niet echt bij de film past.

Ontwerp 2

Bij het tweede ontwerp heb ik het font aangepast met een 3d achtig effect, omdat de echte poster van de film ook zo een effect gebruikt. Verder moet ik nog wat gaan werken aan de achtergrond omdat het nog een beetje simplistisch is.

Ontwerp 3

Bij het derde ontwerp heb ik de feedback gekregen om te gaan werken met wat verschillende filters, ik heb uiteindelijk dit x-ray achtige filter gebruikt.

Ontwerp 4

Het laatste ontwerp wat ik heb gemaakt vind ik veel mooier dan ontwerp drie daarom heb ik gekozen om er geen filter overheen te zetten. uiteindelijk heb ik hier geen feedback op gekregen omdat ik de poster zo mooi vind.

- Berry Sanders

Poster Curious

Beschrijving:

Foto 1

Toen ik de eerste foto aan het bewerken was wilde ik iets curious maken, dus heb ik gekozen voor een rubix cube die opgelost moet worden. Ik heb daarna een zwart wit filter er overheen gezet en zo de kleur geel laten uitspringen.

Foto 2

Bij de tweede foto heb ik een plaatje gepakt van iemand die uit een grot klimt, ik heb vervolgens wat vleermuizen toegevoegd die uit de grot vliegen om het een beetje mysterieus te maken.

Poster Danger

Beschrijving:

Foto 1

Bij foto een wilde ik iets gevaarlijks afbeelden, dus heb ik gekozen voor de classic jaws poster. Ik heb het daarna zwart wit gemaakt en het water een rode kleur gegeven als een indicatie voor gevaar.

Foto 2

Bij de tweede foto heb ik een achtergrond gepakt van een auto die uit brand, tenslotte heb ik een hoofdpersoon uit de serie "The Punisher" uitgeknipt en voor de auto gezet. Verder heb ik het schedel op zijn torso rood gemaakt om gevaar aan te duiden.

Reflectie

Wat heb ik geleerd?

Ik heb tijdens de keuze mediatechnieken veel geleerd, zo heb ik onder andere geleerd hoe ik met Photoshop, Illustrator, After effects en Premier Pro moet werken. Ook heb ik er veel plezier uitgehaald omdat ik het ook leuk vind om te doen.


Wat doe ik de volgende keer anders?

De volgende keer zou ik niet veel anders doen, omdat ik vind dat ik de keuze mediatechnieken goed heb gemaakt. Het enige wat ik vaker moet doen is feedback vragen aan docenten, omdat ik de afgelope tijd de meeste feedback heb ontvangen van mijn groepsgenoten.
