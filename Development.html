Text Animation

Beschrijving:

Ik heb hier een kleine text animatie gemaakt met javascript, ik heb hierbij vanilla javascript gebruikt omdat ik even terug wilde gaan naar de basics van javascript.

In mijn javascript code gebruikt ik een for loop om bij elke letter een span er omheen te zetten, nu dat er om elke letter een span staat kan ik makkerlijker per letter een animation toevoegen.

Uiteindelijk heb ik er niet voor gekozen om deze animatie op mijn website te plaatsen omdat het anders wel erg druk wordt met de verschillende animaties, ook omdat ik al een javascript library gebruikt voor animaties.
