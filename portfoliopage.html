Inspiratie Bronnen

Beschrijving:

Ik ben op internet gaan zoeken naar een aantal designs van websites. Uiteindelijk had ik deze twee ontwerpen gevonden die mij erg aanspraken. Ik heb daarna een combinatie gemaakt van beide ontwerpen, zo heb ik de golf die onder in beeld staat, bij mijn ontwerp laten overvloeien naar een witte achtergrond zodat het niet opvalt dat dit een plaatje is.

Verder heb ik de layout hetzelfde gelaten zoals het ontwerp hierboven, ik heb natuurlijk wel gewoon mijn eigen informatie en social media erin verwerkt.

Iteraties

Beschrijving:

Het linkse plaatje is het eerste ontwerp dat ik had gemaakt voor de website, maar omdat ik het toch niet helemaal perfect vond heb ik het wat aangepast door er nog een witte golf in te zetten. Ik heb de rest van de pagina onder de landings page een witte achtergrond gegeven zodat het mooi overloopt.


Beschrijving:

Bij het linkse plaatje had ik iframe gebruikt om mijn pdf bestanden te laten zien. Al snel kreeg ik te horen dat dit niet echt gebruiksvriendelijk is en niet zo mooi, daarom heb ik het veranderd naar hoe het eruit ziet in het rechtste plaatje. Ik heb er ook een button bij gedaan met de link naar dat desbetreffende pdf bestand, er opent dan een apart tablad zodat je het bestand makkelijker kan inzien.

Reflectie

Wat heb ik geleerd?

Tijdens het maken van mijn portfolio heb ik wat kleine dingen geleerd, bijvoorbeeld het gebruiken van javascript libraries en animations. Verder wist ik al wel veel van HTML en Css omdat ik op mijn vorige opleiding hier ook al mee heb gewerkt.


Wat doe ik de volgende keer anders?

De volgende keer zou ik niet veel dingen anders doen, omdat ik vind dat ik mijn portfolio prima heb gemaakt. Het enige wat ik de volgende keer anders wil doen is een planning maken om zo doelen voor mijzelf te stellen die ik af wil hebben.
